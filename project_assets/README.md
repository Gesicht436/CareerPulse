# Project Assets (`project_assets/`)

The central repository for project documentation, academic reports, and presentation materials.

---

## 1. Key Responsibilities

### **Collective Team (Mayank, Harsh, Ankit, Abhinav 08, Abhinav 285)**

- **Academic Reporting:**
  - Mid-semester and End-semester technical reports.
  - Formatting in LaTeX/Markdown for professional submission.
- **Documentation:**
  - **API Documentation:** Maintaining Swagger/Redoc exports.
  - **User Manuals:** Creating guides for both job seekers and recruiters.
- **Presentations:**
  - Developing slide decks for project reviews.
  - Recording and editing demo videos of the platform.
- **Asset Management:** Storing branding, icons, and diagrams (Excalidraw/LucidChart).

---

## 2. Key Feature Requirements

1. **Single Source of Truth:** Ensuring all documentation is kept in sync with the codebase (e.g., auto-generating API docs from the FastAPI code).
2. **Versioned Reports:** Maintaining a history of technical reports to track the project's evolution.
3. **Asset Organization:** Clear folder structure for images, diagrams, and video demos to ensure they are easily accessible for presentations.
4. **Automated Doc-Checking:** Linting for Markdown files to ensure consistent formatting across all manuals.
